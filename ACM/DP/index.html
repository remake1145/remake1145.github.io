# B题意

# B过程

设定：dp[i][j]dp[i][j]dp[i][j]为在(i,j)(i,j)(i,j)处是否先手必胜。是则为1，否则为0.

  3. 因此，可以先按照aija_{ij}a​ij​​降序排序各点，之后对满足在(i,j)(i,j)(i,j)曼德拉距离m外的点且aij<axya_{ij}<a_{xy}a​ij​​<a​xy​​的点转移。 转移方程： dp[i][j]=1⊕dp[x][y]dp[i][j]=1\oplus dp[x][y]dp[i][j]=1⊕dp[x][y]，条件如上。

# 矩阵快速幂 DP

# K题意

n个点m个边，有k个扬子鳄按2，3，4周期出现在某时刻的某点，该点此时无法访问，求从给定点出发到给定终点的方案数。

# K过程

从离散数学的相关知识可以知道，通过矩阵可以求出在任意给定个时刻中的从某点出发到另一点的通路数。同时，由于每条扬子鳄周期出现，为2，3，4，所以取最小公倍数12为矩阵周期，每个矩阵都处理一遍无法到达的点，之后进行矩阵快速幂直接求出。 具体而言，先建立矩阵，然后求出1-12幂次的矩阵，之后求出output矩阵,然后对output矩阵进行[k12][\frac{k}{12}][​12​​k​​]幂次，之后补上由于相除而缺少的项即可。
